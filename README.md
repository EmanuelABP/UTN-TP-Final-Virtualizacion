## Cómo usar

1. Descargar el archivo `ram.py`.  
2. Ejecutar el script en la terminal de la máquina virtual con Python 3:  
   ```bash
   python3 ram.py
   ```
3. Seguir las instrucciones en pantalla para ingresar datos y ver resultados.
